# NyanCapsule (ニャンカプセル) 🐱✨

きみと猫の魔法の時間を、動画と対話で記録する日記アプリ。

## 機能
- **AI猫との対話日記**: あなたの猫ちゃん（AI）が日記に返信してくれます。
- **動画ダイジェスト**: 日々の動画クリップをRemotionで美しくまとめます。
- **プレミアム・グラスモーフィズム・デザイン**: 温かみのあるモダンなUI。

## 開発状況
- [x] プロジェクト初期化 (Next.js)
- [x] デザインシステム構築 (Vanilla CSS + Glassmorphism)
- [x] チャット形式の日記UI (マックボイス付き)
- [ ] Remotion統合 (動画生成エンジン)
- [ ] リアルAI (LLM) 連携

## 使い方
1. `npm install`
2. `npm run dev`
3. http://localhost:3000 にアクセス
